# Prompts de Imagem Utilizados no Projeto

Este documento contém os prompts refinados utilizados para gerar os recursos visuais do projeto **Canary Circuit**.

Os prompts foram melhorados para ficarem mais claros, organizados e consistentes com a proposta visual do podcast. As principais melhorias foram: correção do inglês, organização das descrições, padronização do estilo visual e melhor separação entre contexto, objetivo e prompt final.

---

## Resumo dos Prompts

| Prompt   | Objetivo                           | Estilo visual                         |
| -------- | ---------------------------------- | ------------------------------------- |
| Prompt 1 | Criar o apresentador do podcast    | Realista, cinematográfico e esportivo |
| Prompt 2 | Criar o mascote Canarinho robótico | Retrô, pixel art e videogame          |
| Prompt 3 | Criar a capa do episódio           | Futurista, tecnológico e profissional |

---

## Prompt 1 — Apresentador do Podcast

| Campo             | Descrição                                               |
| ----------------- | ------------------------------------------------------- |
| Objetivo          | Criar uma imagem realista do apresentador do podcast    |
| Referência visual | Futebol brasileiro e corte inspirado no Ronaldo de 2002 |
| Formato           | 1:1                                                     |
| Estilo            | Fotorealista, cinematográfico e detalhado               |

Este prompt foi melhorado para criar uma imagem realista do apresentador do podcast, mantendo a referência ao futebol brasileiro e ao visual inspirado no Ronaldo de 2002. Também foram organizados os detalhes de iluminação, câmera e qualidade visual.

```text
A cheerful football podcaster inspired by Brazil’s football culture, wearing the Brazilian national team jersey, with the iconic 2002 Ronaldo-style haircut. He is smiling, wearing studio headphones and speaking into a professional microphone. Cinematic composition, realistic facial expression, detailed fabric texture, immersive sports media atmosphere.

Camera settings: f/8, 1/100s, ISO 100. Complex lighting, backlighting, dynamic cinematic lighting, global illumination. Ultra-high-resolution, photorealistic, hyper-detailed, realistic color grading, ultra-realistic photography, sharp focus, 16K quality. Square format, 1:1.
```

## Prompt 2 — Canarinho Robotico

| Campo             | Descrição                                      |
| ----------------- | ---------------------------------------------- |
| Objetivo          | Criar uma versão robótica do mascote Canarinho |
| Referência visual | Mascote da Seleção Brasileira                  |
| Formato           | 1:1                                            |
| Estilo            | Retrô, pixel art, 8 bits e videogame           |

Este prompt foi ajustado para deixar mais claro que o mascote Canarinho deve ter uma aparência robótica, mas ainda amigável e reconhecível. A estética retrô também foi melhor organizada para gerar uma imagem com estilo mais consistente.

```text
Create a robotic version of the Brazilian national team’s “Canarinho” mascot, flying through a football stadium. The mascot should have a futuristic mechanical design while still looking friendly and recognizable. Retro video game style, isometric perspective, pixel-art aesthetic, low-resolution look, down-angle view, 8-bit visual language, Game Boy-inspired color palette, retro sprite style, playful and nostalgic atmosphere. Square format, 1:1.
```

## Prompt 3 — Imagem de Capa

| Campo              | Descrição                                           |
| ------------------ | --------------------------------------------------- |
| Objetivo           | Criar a capa de um episódio do podcast              |
| Elemento principal | Vinícius Júnior com a bola Trionda                  |
| Formato            | 1:1                                                 |
| Estilo             | Futurista, tecnológico, esportivo e cinematográfico |

Este prompt foi refeito para parecer uma capa profissional de episódio de podcast. A descrição foi organizada para destacar Vinícius Júnior, a bola Trionda e os elementos tecnológicos ligados ao futuro do futebol, como gráficos digitais, câmera no árbitro e drone no estádio.

```text
Create a square cover image for a football podcast episode. Show Vinícius Júnior stepping on the official 2026 World Cup ball, Trionda, in the center of the composition. The scene should include strong technology-related elements that represent the future of football: digital overlays, tactical data graphics, motion tracking visuals, futuristic interface elements, a referee wearing a body camera, and a drone flying above the stadium.

The background should be a modern football stadium with dramatic cinematic lighting, energetic atmosphere, and a futuristic sports broadcast feel. The overall style should be photorealistic, ultra-detailed, high-end, and visually striking, like a premium podcast or sports documentary cover.

Include the podcast title clearly in the design:
"How 2026 innovations could reshape the game"

Use clean typography, balanced composition, and a professional cover-art style. Square format, 1:1.
```

## Observação

Os prompts foram escritos em inglês para facilitar o uso em ferramentas de geração de imagem e para manter o projeto mais acessível em um contexto internacional.
