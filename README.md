# Podcast AI

Projeto de podcast criado com o apoio de ferramentas de Inteligência Artificial, utilizando prompts estruturados para gerar roteiro, imagens, narração e edição final.

## Objetivo

O objetivo deste projeto é demonstrar como ferramentas de IA podem ser utilizadas na criação de um podcast completo, desde a concepção do tema até a produção do áudio final.

O podcast aborda as novas tecnologias presentes na Copa do Mundo de 2026 e como essas inovações podem impactar o futuro do futebol, incluindo possíveis mudanças no Brasileirão.

## Tecnologias utilizadas

* ChatGPT
* Gemini / Nano Banana 2
* ElevenLabs
* CapCut
* Notion

## Como foi feito

1. **Roteiro**
   O roteiro do podcast foi criado com o auxílio do ChatGPT, utilizando prompts detalhados para definir tema, estrutura, linguagem e público-alvo.

2. **Imagens**
   As imagens do projeto foram geradas com o Gemini / Nano Banana 2, seguindo a identidade visual proposta para o podcast.

3. **Áudio**
   A narração foi gerada com o ElevenLabs, utilizando voz artificial para simular a apresentação do podcast.

4. **Edição**
   O áudio foi tratado no CapCut, onde também foram adicionados efeitos sonoros e música de fundo.

5. **Organização**
   O projeto foi documentado e organizado no Notion.
